Get Bicycle Racer Information

    fetch('/api/bicycle-racers/{name}')
    
    Example Response: {
      "riderName": "Wout Van Aert",
      "country": "Belgium",
      "team": "Team Jumbo-Visma",
      "age": 27,
      "height": "1.84 m",
      "weight": "78 kg"
    }

Get Race History for Bicycle Racer

fetch('/api/bicycle-racers/{name}/history')

Example Response: [
{
    "raceName": "paris-roubaix",
    "date": "09_April_2023",
    "position": "1"
},
{
    "raceName": "paris-roubaix",
    "date": "17_April_2022",
    "position": "9"
}, etc...]
