Get Race History Of One Racer

        fetch('/api/bicycle-racers/{name}/history')
        
        Example Response: [
        {
        "raceName": "paris-roubaix",
        "date": "09_April_2023",
        "position": "1"
        },
        {
        "raceName": "paris-roubaix",
        "date": "17_April_2022",
        "position": "9"
        }, etc...]


Get Race History Of One Racer Ranked By Position

        fetch('/api/bicycle-racers/{name}/rankedHistory')
        
        Example Response:  {
        "raceName": "paris-roubaix",
        "raceDate": "09_April_2023",
        "position": "1"

        {

        }, etc...]
